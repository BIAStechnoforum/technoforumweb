  • Code Wars

    Coding is one of the things that is easy to do. Even if you get stuck you can google the code for a solution. There is no fun in that. The fun comes when you compete against the best to find the most optimized solutions. Coding becomes interesting, cause in a limited time you got to find the optimized solution without the help of the internet. One can say we can do that by ourselves, but there is no fun in taking part in a competition where one cannot be defeated. Coding under pressure and testing how well you can implement an algorithm you learned to come in handy in the workplace as well as in a competition.

  • Gaming

    Gaming apps and websites are the second most used applications. Playing a one-player game or playing a multiplayer game with the same person is boring. Hence gaming competitions with other people helps you build up your skills. Gaming requires a good eye and hand coordination that is not everyone's cup of tea.
